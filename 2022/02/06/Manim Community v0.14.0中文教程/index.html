Manim Community v0.14.0中文安装教程

注：本教程仅为个人对Manim Community官网中的的翻译，教程仅供学习

1. 安装：

  1. 依据您的使用情况，下面有几个不同安装方式进行推荐。首先，如果您只是想稍稍玩一下Manim, 对于探索Manim库而言，不需要在本地安装的浏览器上的交互式笔记是一种非常简单的方式。链接在这(https://try.manim.community )，通过这个链接就可以得到我们的交互式引导。

  2. 另外，如果您倾向于使用Manim去做动画项目，我们推荐安装本地安装（无论是您系统上的Python，还是通过Docker都可以）。

    注意：

    Manim有很多版本。这个说明书仅仅只是为社区版编辑的。如果您不确定需要安装哪个版本的manim，那么可以通过这个链接(https://docs.manim.community/en/stable/installation/versions.html)进行了解 manim的不同版本的区别

2.安装到本地：

  1. Manim 是一个Python库，并且它可以通过pip命令进行安装。然后，为了让Manim合适的工作，在首次安装一些额外的系统依赖是有必要的。下面有些你需要遵循的操作系统特定的指示。

    Manim到目前为止只支持3.7到3.9版本的Python。

  2. 注意事项：

    依照您特别的设置，这个安装过程可能会有些轻微不同。请确保您已经小心地尝试遵循下述的步骤，但是如果你遇到了一些麻烦，我们很乐意进行帮忙，可以加入我们的Discord（https://discord.com/invite/bYCyhM9Kz），或者直接在Github开始一个新的讨论。

  3. window安装

    使用一个包管理（像Chocolatey或者Scoop）安装Manim是最简单的方式。如果你害怕编辑您的环境变量，或者进行可能的手动安装。事实上，如果你已经安装了Python(3.7-3.9)，这中方式是最简单的方式让所有的文件启动并运行。

    如果你选择了一种包管理，请遵循他们的安装指南（Chocolatey或者Scoop的）使得他们可以在您的系统上可以获得。

    1. 需要的依赖

      Manim需要一个最近的Python版本（3.7-3.9）和ffmpeg才能工作。

      Chocolatey

      Manim可以被安装通过简单运行Chocolatey。

      choco install manimce

      就仅此而已，没有更多的步骤需要了。可以继续安装下面的选择性依赖。

      Scoop

      然而，这儿也没有什么独家配方通过直接使用Scoop安装，你可以安装所有的需要文件通过使用下面的命令

      scoop install python ffmpeg

      然后通过运行这些命令就可以安装Manim

      python -m pip install manim

      手动安装

      正如上面所提到的，Manim 需要合适的就近的Python3版本（3.7-3.9）和FFmpeg

      Python：通过这个链接(https://www.python.org/)进行下载安装Python（3.7-3.9）并且遵循它的说明书使得Python安装在你的系统上。

      注意：我们现在已经得到了很多因为从Window 商店中下载的python版本而引起问题的报告了。对此，我们推荐不要下载window商店上面的下载Python。作为替换，请直接点击官网下载。

      FFmpeg：为了安装FFmpeg，你可以从这个链接( https://ffmpeg.org/download.html#build-windows)中得到预编译和即将使用的版本，例如(https://www.gyan.dev/ffmpeg/builds/packages/ffmpeg-5.0-essentials_build.7z)
      推荐使用可以使用的版本，或者您完全知道您正在做上面，您可以替代性地使用来自这里(https://ffmpeg.org/download.html )的源代码，自己对其进行编译。

      在下载完预编译的文件后，解压它，如果你愿意的话，请移动解压后的目录到一些更永久的地方（比如c：\Program Files\）。下一步，编辑环境变量，把bin路径添加到环境变量。最后完成修改并且退出。

      如果你现在打开命令行并且运行$ ffmpeg $,这个命令行应该会被识别。

      在此，可以通过

      python -m pip install manim

      获得安装Manim全部必要的依赖。

    2. 可选择的依赖

      为了确保Manim的交互页面可以使用LaTeX，举例—画数学等式，LaTeX应该也被安装。注意哈，这是一个可选择性的依赖，如果不倾向于使用LaTeX，你也可以不安装它。

      对于window来说，被推荐的LaTeX发行版是MikTeX。你可以通过使用来自MikTeX网站的安装器安装它，或者使用你选择的包管理进行下载

      Chocolatey：

      choco install miktex.install

      Scoop:

      scoop install latex

      如果你担心磁盘空间，这儿也有替代品。像TinyTeX是更小的LaTeX发行版。

      Chocolatey：

      choco install tinytex

      Scoop:

      首先：

      scoop bucket add r-bucket https://github.com/cderv/r-bucket.git

      然后

      scoop install tinytex

      在这种情况下，你将不得不通过tlmgr管理LaTeX的包安装在你的系统。因此，我们只在你已经知道你正在做什么的情况下推荐这个选项。Manim交互可能需要的所有的LaTeX包如下：

      amsmath babel-english cbfonts-fd cm-super ctex doublestroke dvisvgm everysel
fontspec frcursive fundus-calligra gnu-freefont jknapltx latex-bin
mathastext microtype ms physics preview ragged2e relsize rsfs
setspace standalone tipa wasy wasysym xcolor xetex xkeyval

      注意：

      对于Chocolatey 来说，基于更小的LaTeX发行版TinyTeX来说，一个值得投入的

      manim-latex

      包含了这些包。

      如果你使用Chocolatey你可以通过使用

      choco install manim-latex

      得到上面的包。

    3. 同Manim进行工作

      到目前为止，你一个可以安装Manim了。快去前往快速导学去学习如何制作你自己的Manim动画吧！！！
