Subjunctive mood

The basic structure

“Ould”+Have done

  1. Should + have done = 应该做但没有做的事情
  2. Would + have done = 想要做但是没有做的事情
  3. Could + have done = 可以做但是没有做的事情

Conditional sentences with if

时态 if+ 主句
现在或者将来 过去时基础态 would + V原型
过去 过去时完成态 would + have done

Inversion structures

If the first verb in a conditional If-Clause is should, were or had. We can leave out if and put the verb at the start of the clause:

  • If any of this should cost you anything, send me the bill.
  • ==> Should any of this cost you anything, send me the bill
  • It would be embarrassing, if she were to find out the truth
  • ==> It would be embarrassing, were she to find out the truth
  • It I had known you were coming, I’d have baked a cake.
  • ==> Had I known you were coming, I’d have baked a cake.

Other about “Should”

From Cambridge Dictionary

should modal verb (POSSIBILITY)

formal

used when referring to a possible event in the future:

If anyone should ask for me, I’ll be in the manager’s office.

Should you (= if you) ever need anything, please don’t hesitate to contact me.

补充：虚拟语气中的 were

Use were if the state of being you are describing is in no way the current reality. This is true whenever a hypothetical假想 situation is expressed, for example：

例：

  • If Harry were to find all the Horcruxes in time, we would not be forced to submit to Voldemort’s evil will.
  • If I were two-faced, would I be wearing this one?
  • Would you invite me over if I were more polite at the dinner table?

在口语中，在第一人称和第三人称单数时was也可以使用。但有两种情况下，必须用were：

  • If I were you 例： If I were you, I would have continued working until it was done.
  • as it were 例：He is, as it were , a walking dictionary.
